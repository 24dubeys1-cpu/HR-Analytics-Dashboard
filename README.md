# HR Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing employee data to uncover patterns related to employee attrition, retention, salary distribution, and workforce demographics.

The dashboard helps HR teams and business leaders make data-driven decisions by identifying factors that influence employee turnover and overall workforce performance.

---

## 🎯 Objectives

- Analyze employee dataset
- Identify patterns in employee attrition
- Compare attrition across departments and job roles
- Perform correlation analysis on key HR metrics
- Build important HR KPIs
- Create an interactive dashboard for decision-making

---

## 📊 Key Performance Indicators (KPIs)

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate (%)
- Retention Rate (%)
- Average Salary
- Average Years of Experience
- Average Age
